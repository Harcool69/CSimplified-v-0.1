# CSimplified

---

A tiny interpreted programming language written in C.

---

## Features

- Variables
- Conditions
- Printing
- Input handling

---

## Build

```bash
make
```
---

## Run

```bash
./bin/csimplified examples/hello.csimpl
```
---

## Example

```csimplified
print "Hello world"
```
