# CSimplified

---

A tiny interpreted programming language written in C.

---

## Features

- Variables
- Conditions
- Printing
- Input handling

---

## Build

```bash
make
```
---

## Run

```bash
./src/main your_script_path
```
If you want to compile
```bash
cd src #if you are not in src directory
gcc main.c parser.c interp.c utils.c -o main
```
---

## Example

```csimplified
print "Hello world"
```
